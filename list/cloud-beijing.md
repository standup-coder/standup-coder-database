# 北京云计算企业清单

## 1. 主题介绍
本文件列出了北京地区主要的云计算企业。北京作为中国首都和科技创新中心，汇聚了大量云计算领域的龙头企业和创新型公司。百度智能云、字节跳动火山引擎、金山云、青云科技等多家知名云服务商均将总部设在北京。
