# 安全运维工程师职位描述(JD)

## 岗位基本信息
- **岗位名称**：安全运维工程师(Security Operations Engineer)
- **工作性质**：全职
- **工作地点**：北京/上海/深圳/杭州/成都
- **招聘人数**：若干
- **薪资范围**：25K-50K/月(根据经验水平)

## 岗位职责

### 安全体系建设
- 负责企业信息安全体系的规划和建设
- 制定安全策略和防护标准规范
- 部署和维护安全防护设备和系统
- 实施安全监控和威胁检测机制
- 建立安全事件应急响应流程

### 安全运维管理
- 负责网络安全设备的日常运维
- 监控系统安全状态和异常行为
- 处理安全漏洞和风险评估
- 实施安全补丁和版本更新
- 优化安全配置和防护策略

### 安全技术支持
- 为业务部门提供安全技术咨询
- 协助开发团队进行安全代码审查
- 参与安全项目的技术评估
- 分享安全最佳实践和经验
- 推动安全意识培训和文化建设

## 任职要求

### 必备技能
**安全技术**：
- 精通网络安全原理和防护技术
- 熟悉主流安全设备和工具
- 掌握漏洞扫描和渗透测试方法
- 了解安全合规和审计要求
- 熟悉安全事件分析和处置流程

**运维能力**：
- 具备良好的系统运维能力
- 熟悉Linux/Windows系统安全配置
- 具备安全故障排查和解决能力
- 了解网络协议和安全防护机制
- 具备自动化安全运维能力

### 经验要求
- **学历要求**：本科及以上学历，信息安全/计算机相关专业
- **工作经验**：
  - 初级岗位：3-5年安全运维经验
  - 中级岗位：5-8年安全工程经验
  - 高级岗位：8年以上安全专家经验
- **项目经验**：有大型安全体系建设经验

### 加分技能
- CISSP/CISP等安全认证
- 渗透测试和红蓝对抗经验
- 安全自动化工具开发经验
- 云安全架构设计经验
- 开源安全项目贡献经历

- **测试工具**：Nessus + Burp Suite + Metasploit
- **自动化**：Ansible + Python + Shell脚本
- **云安全**：AWS Security + Azure Security Center

- 持续学习，跟上威胁演进

- 定期技术分享和攻防演练
