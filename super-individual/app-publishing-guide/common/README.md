## 为什么需要软著和版号

### 软件著作权（软著）

```bash
# 国内安卓分发必备：
# - 华为、小米、OPPO、vivo、应用宝等
# - 证明 App 归属权
# - 避免侵权纠纷

# 周期：
# - 自己申请：30-50 工作日
# - 加急代理：7-15 天
```

### 版号

```bash
# 国内商业化必需：
# - 游戏类 App 必须
# - 部分内容类 App 需要
# - 新闻/视频类需要

# 周期：
# - 游戏版号：3-6 个月
# - 其他类目：1-3 个月
```

## 软著 vs 版号

| 项目 | 软件著作权 | 版号 |
|------|-----------|------|
| 适用 | 工具/App/游戏 | 游戏/内容 |
| 审批部门 | 版权局 | 出版署/工信部 |
| 周期 | 1-2 个月 | 3-6 个月 |
| 费用 | 免费~2000 | 3000-5000+ |
| 难度 | 较低 | 较高 |
