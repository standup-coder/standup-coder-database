# 嵌入式开发工程师职位描述(JD)

## 岗位基本信息
- **岗位名称**：嵌入式开发工程师(Embedded Software Engineer)
- **工作性质**：全职
- **工作地点**：北京/上海/深圳/杭州/成都
- **招聘人数**：若干
- **薪资范围**：20K-45K/月(根据经验水平)

## 岗位职责

### 嵌入式系统开发
- 负责嵌入式软件的设计、开发和调试
- 开发底层驱动程序和固件
- 实现硬件抽象层和中间件
- 优化系统性能和资源利用率
- 解决硬件兼容性和稳定性问题

### 系统集成测试
- 参与硬件选型和系统架构设计
- 进行系统集成和联调测试
- 编写技术文档和用户手册
- 支持产品生产和质量控制
- 处理客户技术问题和反馈

### 技术研究创新
- 跟踪嵌入式技术发展趋势
- 研究和应用新技术到产品开发
- 优化开发流程和工具链
- 分享嵌入式开发经验和技术
- 参与技术标准制定和专利申请

## 任职要求

### 必备技能
**核心技术栈**：
- 精通C/C++编程语言
- 熟悉ARM/MIPS等处理器架构
- 掌握RTOS实时操作系统
- 了解硬件电路和接口协议
- 熟悉嵌入式调试工具和方法

**开发能力**：
- 具备良好的系统设计能力
- 熟悉嵌入式开发流程
- 具备硬件调试和问题定位能力
- 了解通信协议和网络编程
- 具备代码优化和性能调优能力

### 经验要求
- **学历要求**：本科及以上学历，电子工程/计算机相关专业
- **工作经验**：
  - 初级岗位：3-5年嵌入式开发经验
  - 中级岗位：5-8年嵌入式系统开发经验
  - 高级岗位：8年以上嵌入式专家经验
- **项目经验**：有完整的嵌入式产品开发经验

### 加分技能
- 物联网(IoT)开发经验
- Linux内核开发经验
- 硬件设计基础
- 自动化测试经验
- 开源项目贡献经历

- **调试工具**：JTAG + Logic Analyzer
- **版本控制**：Git + SVN
