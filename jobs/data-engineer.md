# 数据工程师职位描述(JD)

## 岗位基本信息
- **岗位名称**：数据工程师(Data Engineer)
- **工作性质**：全职
- **工作地点**：北京/上海/深圳/杭州/成都
- **招聘人数**：若干
- **薪资范围**：20K-55K/月(根据经验水平)

## 岗位职责

### 数据平台建设
- 设计和构建企业级数据平台架构
- 开发和维护ETL数据管道
- 构建和优化数据仓库及数据湖
- 实现数据质量管理体系建设
- 保障数据平台稳定性和性能

### 数据处理开发
- 开发批处理和流处理数据作业
- 设计实时数据处理和分析系统
- 优化数据存储和查询性能
- 实现数据安全和权限管控
- 监控数据流程和异常告警

### 业务支持协作
- 与数据科学家协作提供数据支持
- 为业务部门提供数据接口和服务
- 参与数据需求分析和技术方案设计
- 指导业务人员数据使用方法
- 推动数据驱动的业务决策

## 任职要求

### 必备技能
**核心技术栈**：
- 精通Python/Java/Scala至少一门编程语言
- 熟悉大数据处理框架(Hadoop/Spark/Flink)
- 掌握SQL和至少一种NoSQL数据库
- 熟悉数据仓库建模理论和实践
- 掌握Git版本控制和敏捷开发流程

**工程能力**：
- 具备良好的系统设计和架构能力
- 熟悉分布式系统原理和实践
- 具备性能调优和故障排查能力
- 了解数据安全和隐私保护要求
- 具备自动化运维和监控能力

### 经验要求
- **学历要求**：本科及以上学历，计算机/数学/统计学相关专业
- **工作经验**：
  - 初级岗位：2-4年数据工程相关经验
  - 中级岗位：4-7年数据工程经验
  - 高级岗位：7年以上数据平台建设经验
- **项目经验**：有大型数据平台或数据仓库建设经验

### 加分技能
- 云数据平台使用经验(AWS/Azure/GCP)
- 实时流处理技术(Kafka Streams/Flink)
- 数据治理和元数据管理经验
- 容器化和微服务架构经验
- 开源大数据项目贡献经历


### 管理发展路径
```
Senior Data Engineer → Data Team Lead → Data Platform Manager
```
