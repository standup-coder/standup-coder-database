## Notion（超级个体首选）

### 为什么选 Notion

```bash
# 1. All-in-One
#    - 笔记
#    - 任务管理
#    - 数据库
#    - 日历
#    - Wiki

# 2. 免费版够用
#    - 无限页面
#    - 10MB 上传限制
#    - 1000 个 blocks

# 3. 模板丰富
#    - 有大量免费模板
#    - 适合各种场景
```

### Notion 快速入门

#### 注册和使用

```bash
# 1. 访问 https://www.notion.so/
# 2. 使用邮箱或 Google 账号注册
# 3. 开始创建页面

# 基础操作：
# - /page：创建新页面
# - /todo：创建待办列表
# - /table：创建数据库
# - /calendar：创建日历
```

#### 超级个体的 Notion 模板

```bash
# 推荐创建的页面：
# 1. 主页/仪表盘
#    - 今日待办

# 2. 产品路线图
#    - 功能列表
#    - 优先级
#    - 状态

# 3. 用户反馈收集
#    - 反馈数据库
#    - 来源分类
#    - 状态跟踪

# 4. 内容创作库
#    - 文章大纲
#    - 发布计划
#    - 社交媒体内容

# 5. 财务管理
#    - 收入记录
#    - 支出记录
#    - 月度总结
```

#### Notion 数据库示例

```bash
# 创建产品路线图数据库：

# 1. 新建页面
# 2. 输入 /table 并选择 "Table - Full Page"
# 3. 添加属性：
#    - 名称（Title）
#    - 状态（Select）：规划中/开发中/已完成
#    - 优先级（Select）：高/中/低
#    - 截止日期（Date）
#    - 备注（Text）

# 4. 添加视图：
#    - Board（看板视图）
#    - Calendar（日历视图）
#    - Timeline（时间线视图）
```

| Notion | 免费 | 云 | All-in-One |

### Obsidian 推荐

```bash
# 为什么选 Obsidian：
# 1. 本地存储（数据完全属于你）
# 2. Markdown 支持好
# 3. 双链功能（类似 Roam Research）
# 4. 插件丰富
# 5. 免费版功能全

# 同步方案：
# - 使用 iCloud（Mac 用户）
# - 使用 Git（免费）
# - 使用 Obsidian Sync（付费 $8/月）
```

## 时间管理

### 时间追踪工具

#### Toggl Track（推荐）

```bash
# 1. 访问 https://toggl.com/
# 2. 注册免费账号
# 3. 下载桌面/手机 App

# 使用方法：
# 1. 点击 "Start" 开始计时
# 2. 选择项目或输入描述
# 3. 点击 "Stop" 结束
# 4. 查看周/月报告

# 免费版功能：
# - 无限项目
# - 详细报告
# - 导出功能
```

#### 超级个体的替代方案

```bash
# 如果不想专门工具：
# 1. 苹果自带"计时器" App
# 2. 在 Notion 中创建简单时间表
# 3. 纸笔记录
```

### 番茄工作法

```bash
# 工具：Forest（App Store / Play Store）

# 设置：
# 1. 下载 Forest
# 2. 设置专注时长（默认 25 分钟）
# 3. 设置休息时长（默认 5 分钟）

# 使用：
# 1. 开始专注，种一棵树
# 2. 中途退出，树会枯死
# 3. 累积金币可解锁新树种

# 优点：游戏化，容易坚持
# 缺点：付费版才有完整功能
```
