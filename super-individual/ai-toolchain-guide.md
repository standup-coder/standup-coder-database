### 2.3 提示词工程核心技巧

**结构化提示词模板**：

```
# 角色设定
你是一位【具体角色】，拥有【相关经验】背景。

# 任务描述
请帮我【具体任务】，要求：
1. 【要求1】
2. 【要求2】
3. 【要求3】

# 输出格式
请按以下格式输出：
- 【格式要求】

# 参考信息
以下是相关背景信息：
【粘贴参考内容】

# 约束条件
- 语言：【中文/英文】
- 长度：【字数范围】
- 风格：【专业/轻松/学术等】
- 受众：【目标读者】
```

**高阶技巧**：
- **思维链（CoT）**：要求模型"逐步思考"
- **Few-shot**：提供2-3个示例引导输出格式
- **自检机制**：让模型自我审查并改进输出
- **角色链**：多轮对话逐步深化，每轮聚焦一个维度
- **对比生成**：让模型生成多个版本供选择

```

├── ChatGPT → 分析热门话题趋势
├── 生成选题列表和内容大纲
└── 竞品内容分析和差异化定位

├── Claude → 生成初稿（质量最高）
├── ChatGPT → 扩展内容、补充案例

├── AI → 生成配图文案

```

### 4.3 AI内容创作注意事项

- **避免纯AI生成**：平台算法可能降低AI内容的推荐权重
- **加入个人经验**：真实经历和独特观点是差异化关键
- **事实核查**：AI可能生成不准确的信息，务必核实
- **版权意识**：AI生成内容的版权归属尚不明确
- **风格一致**：保持个人写作风格的一致性
