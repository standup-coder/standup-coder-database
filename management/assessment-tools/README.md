# 管理评估工具

## 概述
本目录提供技术管理能力的评估体系和诊断工具，帮助管理者识别优势、发现短板并制定改进计划。

## 核心评估体系

### 📊 能力评估框架
- `technical-management-assessment.md` - 技术管理能力专项评估
  - 管理能力核心维度（战略、团队、执行）
  - 数字化领导力评估
  - 五级管理能力阶梯
  - 专项评估指标体系

## 评估维度详解

### 战略领导力
- 技术愿景制定和传达
- 长期技术规划能力
- 组织变革推动能力
- 行业趋势洞察力

### 团队领导力
- 团队建设和人才培养
- 激励和赋能下属
- 冲突处理和协调
- 团队文化建设

### 执行领导力
- 项目管理和交付
- 资源协调和分配
- 质量管控和风险
- 效率优化和改进

## 评估工具类型

### 定量评估
- 360度评估体系
- 管理效能量化指标
- 团队健康度评估
- 个人管理成效指标

### 定性评估
- 情景模拟评估
- 行为面试评估
- 案例分析方法
- 同行评议机制

## 使用建议

### 自我评估
1. 使用评估问卷进行初步自评
2. 对照能力模型识别优势和短板
3. 制定个人发展改进计划

### 组织应用
1. 建立定期评估机制
2. 结合评估结果制定培训计划
3. 跟踪改进效果和进展

### 评估频率建议
- **个人自评**：季度一次
- **正式评估**：半年一次
- **360度评估**：年度一次
- **发展跟踪**：月度回顾
