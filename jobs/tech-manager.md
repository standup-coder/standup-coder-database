# 技术经理职位描述(JD)

## 岗位基本信息
- **岗位名称**：技术经理(Technical Manager / Engineering Manager)
- **工作性质**：全职
- **工作地点**：北京/上海/深圳/杭州/成都
- **招聘人数**：若干
- **薪资范围**：35K-80K/月(根据经验水平)

## 岗位职责

### 团队管理与建设
- 负责技术团队的组建、培养和日常管理
- 制定团队目标并推动目标达成
- 负责团队成员的绩效评估和职业发展
- 建设团队技术文化和工作氛围
- 协调跨团队协作和资源分配

### 项目管理交付
- 主导技术项目的需求分析和方案设计
- 制定项目计划并把控进度和质量
- 识别和管理项目风险
- 组织技术评审和代码审查
- 推动项目按时高质量交付

### 技术决策与架构
- 参与系统架构设计和技术选型
- 把控技术方向和代码质量
- 解决团队技术难题和阻塞问题
- 推动技术债务清理和系统重构
- 建立和完善技术规范和流程

### 业务价值实现
- 深入理解业务需求和用户痛点
- 推动技术与业务的深度融合
- 评估技术投入产出比
- 支持产品和技术创新
- 向上汇报技术进展和风险

## 任职要求

### 必备技能
**技术能力**：
- 扎实的技术功底，精通至少一个技术领域
- 具备系统架构设计和评审能力
- 了解前后端、数据库、DevOps等全栈技术
- 熟悉敏捷开发和项目管理方法论
- 具备技术风险识别和决策能力

**管理能力**：
- 具备团队管理和人员培养经验
- 熟悉目标管理和绩效评估体系
- 具备优秀的沟通协调能力
- 善于处理冲突和解决问题
- 具备向上管理和横向协作能力

**领导力**：
- 能够激励团队并建立信任
- 具备技术影响力和说服力
- 善于培养人才和梯队建设
- 具备战略思维和业务敏感度
- 能够在压力下保持冷静决策

### 经验要求
- **学历要求**：本科及以上学历，计算机相关专业
- **工作经验**：
  - 初级管理岗：5-8年技术经验，1-3年管理经验
  - 中级管理岗：8-12年技术经验，3-5年管理经验
  - 高级管理岗：12年以上经验，5年以上大型团队管理经验
- **团队规模**：有管理5-30人技术团队的经验
- **项目经验**：有主导大型技术项目交付的成功案例

### 加分技能
- 大型互联网公司或知名技术团队管理经验
- 技术社区影响力（演讲、博客、开源）
- 多团队/多地域管理经验
- 技术预算和资源规划经验
- MBA或管理类培训经历

## 薪酬福利

### 薪资结构
- **基本工资**：根据经验35K-80K/月
- **管理津贴**：团队管理额外津贴
- **项目奖金**：项目成功交付奖励
- **年终奖**：根据团队和部门绩效
- **股票期权**：核心管理岗位享有期权激励

### 福利待遇
- 五险一金 + 补充高端医疗保险
- 弹性工作时间 + 远程办公支持
- 年度体检 + 带薪年假
- 管理培训 + 领导力发展项目
- 团建活动 + 节日福利

## 职业发展

### 管理发展路径
```
Tech Manager → Senior Engineering Manager → Director of Engineering
          ↓
   VP of Engineering → CTO
```

### 技术+管理双轨路径
```
Engineering Manager → Principal Engineer / Distinguished Engineer
```

### 专业细分方向
- **研发管理**：产品研发团队管理
- **工程效能**：DevOps/SRE/效能团队管理
- **AI技术管理**：算法/数据团队管理
- **架构管理**：架构师团队和技术委员会
