# 生产力工具链指南

## 1. 概述

高效的工具链是超级个体的基础设施。本指南系统梳理超级个体日常运营中所需的核心生产力工具，覆盖项目管理、知识管理、设计工具、自动化工具、协同办公、客户管理等关键领域，帮助个人从业者构建"一人公司"级别的数字化运营体系。

### 3.2 知识管理体系搭建

```
个人知识管理（PKM）体系：

收集层（信息输入）：
├── RSS订阅 → Inoreader / Feedly
├── 稍后阅读 → Cubox / Pocket
├── 微信收藏 → 微信读书 / 印象笔记
├── 网页剪藏 → Notion Web Clipper
└── AI信息源 → Perplexity / ChatGPT

整理层（信息处理）：
├── 笔记工具 → Obsidian / Notion
├── 标签体系 → 统一标签规范
├── 知识卡片 → 原子化笔记
└── 定期回顾 → 周度/月度整理

输出层（知识应用）：
├── 技术博客 → 公众号/CSDN
├── 项目文档 → GitHub Wiki
├── 视频脚本 → B站/抖音
└── 课程内容 → 知识付费产品

存档层（长期保存）：
├── 本地备份 → Git仓库
├── 云端备份 → 云存储
└── 重要归档 → 专门文件夹
```
