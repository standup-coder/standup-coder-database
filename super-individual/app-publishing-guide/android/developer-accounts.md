### Google Play 详细注册

```bash
# Google Play 是 Android 官方应用商店
# 是海外发行的必备渠道
```

#### 第一步：注册 Google 账号

```bash
# 1. 访问 https://play.google.com/console/
# 2. 点击 "Sign in" → "Create account"
# 3. 选择 "For work" 或 "Personal"
# 4. 填写信息完成注册
```

#### 第二步：注册开发者账号

```bash
# 1. 访问 https://play.google.com/console/developer
# 2. 使用 Google 账号登录
# 3. 阅读并同意 "Developer Distribution Agreement"
# 4. 支付注册费用：$25（一次性）

# 支付方式：
# - Visa/MasterCard 信用卡
# - 借记卡
# - Google Play 礼品卡（需要美国地址购买）

# 支付时可能遇到的问题：
# - 信用卡被拒：尝试用 PayPal
# - 地区限制：可能需要 VPN + 美国 IP
```

#### 第三步：完善开发者资料

```bash
# 1. 填写开发者名称（将显示在 Play Store）
# 2. 填写联系方式（邮箱）
# 3. 填写联系电话
# 4. 开发者账号注册完成
```

### 详细注册步骤

#### 第一步：访问开发者平台

```bash
# 1. 访问 https://developer.huawei.com/consumer/cn/
# 2. 点击右上角 "登录"
# 3. 如果没有账号，点击 "注册"
```

#### 第二步：选择开发者类型

```bash
# 个人开发者：
# 适合独立开发者
# 需要材料：
# - 身份证（正反面照片）
# - 手机号码
# - 邮箱

# 企业开发者：
# 适合公司或团队
# 需要材料：
# - 营业执照（影印件）
# - 法人身份证（正反面）
# - 对公账户（用于验证）
# - 联系人身份证
```

#### 第三步：填写认证信息

```bash
# 个人认证：
# 1. 填写姓名、身份证号
# 2. 上传身份证照片（正反面）
# 3. 进行人脸识别（部分情况）
# 4. 等待审核（1-3 天）

# 企业认证：
# 1. 填写公司信息
# 2. 上传营业执照
# 3. 填写法人信息
# 4. 对公账户打款验证（几分钱验证）
# 5. 等待审核（3-7 天）
# 6. 认证费用：¥500
```

#### 第四步：审核通过后创建应用

```bash
# 1. 登录开发者平台
# 2. 进入 "应用服务" → "应用市场"
# 3. 点击 "添加应用"
# 4. 填写应用基本信息
# 5. 提交审核
```

## 常见问题

**Q: 企业账号和个人账号区别？**
A: 企业账号可显示公司名，部分渠道有首发加成。

**Q: 软著申请需要多久？**
