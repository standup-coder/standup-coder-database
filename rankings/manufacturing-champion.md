# 国家级制造业单项冠军企业清单

## 1. 主题介绍
本文件汇总了工业和信息化部认定的国家级制造业单项冠军企业。制造业单项冠军是指长期专注于制造业某一细分产品领域，生产技术或工艺国际领先，单项产品市场占有率位居全球或全国前列的企业。截至2025年，工信部已认定九批制造业单项冠军企业，累计超过1,500家（含单项冠军示范企业和单项冠军产品）。

## 2. 认定标准

| 类别 | 认定条件 |
|------|----------|
| 单项冠军示范企业 | 专注细分领域10年以上，全球市场占有率前3或国内第1 |
| 单项冠军产品 | 专注细分领域3年以上，全球市场占有率前10或国内前3 |
| 专注度 | 主营产品收入占企业营收70%以上 |
| 创新能力 | 研发费用占比不低于3%，拥有核心自主知识产权 |
| 质量效益 | 产品质量达到国际先进水平，盈利能力行业领先 |
