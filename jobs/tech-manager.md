# 技术经理职位描述(JD)

## 岗位基本信息
- **岗位名称**：技术经理(Technology Manager)
- **工作性质**：全职
- **工作地点**：北京/上海/深圳/杭州
- **招聘人数**：若干
- **薪资范围**：35K-65K/月(根据经验水平)

## 岗位职责

### 团队管理建设
- 负责技术团队的组建、培养和管理
- 制定团队发展规划和人才培养计划
- 建立高效的团队协作机制
- 提升团队技术能力和工作效率
- 处理团队内部沟通和冲突

### 技术决策规划
- 参与公司技术战略制定和实施
- 评估和引入新技术和工具
- 制定技术架构和开发规范
- 审核技术方案和项目计划
- 平衡技术债务和业务需求

### 项目管理协调
- 负责重点项目的技术实施和交付
- 协调跨部门技术资源和需求
- 监控项目进度和质量控制
- 识别和解决项目风险问题
- 向上级汇报项目状态和成果

## 任职要求

### 必备技能
**管理能力**：
- 具备丰富的团队管理经验
- 熟悉敏捷开发和项目管理方法
- 具备优秀的沟通协调能力
- 能够有效激励和培养团队成员
- 具备冲突处理和决策能力

**技术背景**：
- 具备扎实的软件开发技术基础
- 熟悉主流技术架构和开发流程
- 了解前沿技术发展趋势
- 具备系统设计和架构能力
- 有成功的技术项目管理经验

### 经验要求
- **学历要求**：本科及以上学历，计算机相关专业
- **工作经验**：
  - 中级管理：5-8年技术开发经验，2年以上团队管理经验
  - 高级管理：8-12年技术经验，5年以上管理经验
  - 资深管理：12年以上技术经验，8年以上管理经验
- **项目经验**：有大型技术项目管理成功案例

### 加分技能
- MBA或相关管理学位
- PMP/敏捷教练等认证
- 跨文化团队管理经验
- 创业公司管理经验
- 技术布道和演讲经验

## 薪酬福利

### 薪资结构
- **基本工资**：根据经验35K-65K/月
- **管理奖金**：团队绩效额外奖励
- **年终奖**：根据公司业绩和个人贡献

- 弹性工作制 + 远程管理支持
- 年度体检 + 健康管理服务
- 管理培训 + 领导力发展项目
- 团队建设预算 + 商务差旅支持

## 职业发展

### 管理发展路径
```
Technology Manager → Senior Technology Manager → Director of Engineering
                ↓
        VP of Engineering → CTO
```

### 技术管理路径
```
Technical Manager → Technical Director → Chief Technology Officer
```

### 学习成长机会
- 参与公司战略决策和规划
- 接触业界顶尖管理实践
- 与CEO和高管团队协作
- 管理培训和领导力提升
- 行业会议和高端交流

## 应聘方式

### 简历要求
请在简历中重点说明：
- 团队管理经验和成果
- 技术项目成功案例
- 领导力和管理理念
- 跨部门协作经验
- 职业发展规划和期望

### 面试流程
1. **初步沟通**(HR电话)：30分钟基本情况了解
2. **管理面试**(HRBP)：60分钟管理理念交流
3. **技术面试**(技术VP)：90分钟技术管理能力评估
4. **业务面试**(业务负责人)：60分钟业务理解沟通
5. **高管面试**(CEO/CTO)：45分钟战略思维考察
6. **发放Offer**：通过后1-2周内正式发出
