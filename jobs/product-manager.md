# 产品经理职位描述(JD)

## 岗位基本信息
- **岗位名称**：产品经理(Product Manager)
- **工作性质**：全职
- **工作地点**：北京/上海/深圳/杭州/广州
- **招聘人数**：若干
- **薪资范围**：20K-60K/月(根据经验水平)

## 岗位职责

### 产品规划
- 负责产品战略规划，制定产品路线图
- 分析市场需求和竞品动态，挖掘用户痛点
- 定义产品功能优先级，制定迭代计划
- 撰写产品需求文档(PRD)和功能规格说明

### 需求管理
- 收集和分析用户需求，转化为产品功能
- 与业务方沟通，理解业务目标和约束
- 管理需求池，评估需求价值和可行性
- 组织需求评审，确保团队理解一致

### 项目推进
- 协调设计、开发、测试等团队资源
- 跟进产品开发进度，确保按时交付
- 组织产品验收，把控产品质量
- 推动产品上线，监控上线效果

### 数据分析
- 建立产品数据指标体系
- 分析产品数据，发现问题和机会
- 基于数据驱动产品优化决策
- 输出产品数据分析报告

## 任职要求

### 必备技能
**产品能力**：
- 熟练掌握产品需求文档撰写
- 具备用户研究和用户体验设计能力
- 熟悉敏捷开发流程和项目管理方法
- 掌握数据分析工具和方法

**工具技能**：
- Axure/Figma/Sketch 原型设计工具
- Jira/Confluence/禅道等项目管理工具
- SQL 数据分析能力
- 办公软件(Excel/PPT/Word)

### 经验要求
- **学历要求**：本科及以上学历
- **工作经验**：
  - 初级：1-3年产品经验
  - 中级：3-5年产品经验
  - 高级：5年以上产品经验
- **行业经验**：有互联网产品经验优先

### 加分项
- 有成功的互联网产品案例
- 技术背景或数据分析背景
- 熟悉特定行业(电商/金融/教育等)
- 优秀的沟通和跨团队协作能力

## 薪酬福利

### 薪资结构
- **基本工资**：20K-60K/月
- **绩效奖金**：年度1-6个月薪资
- **股票期权**：核心产品岗位享有期权
- **项目奖金**：产品成功上线奖励

### 福利待遇
- 五险一金 + 补充医疗保险
- 弹性工作时间
- 年度体检 + 带薪年假
- 产品培训 + 行业交流机会

## 职业发展

### 发展路径
```
产品专员 → 产品经理 → 高级产品经理 → 产品总监 → VP of Product
                    ↓
            细分领域专家(增长/商业化/平台)
```
