# IT岗位信息清单

## 项目介绍
本文件夹收录了全球IT行业各类技术岗位的详细信息，涵盖岗位职责、技能要求、薪资水平、职业发展路径等核心内容。为求职者、HR从业者和技术管理者提供专业、全面的岗位参考信息。

### 1. 软件开发类岗位
- 前端开发工程师
- 后端开发工程师
- 全栈开发工程师
- 移动开发工程师
- 游戏开发工程师
- 嵌入式开发工程师

### 2. 数据技术类岗位
- 数据工程师
- 数据科学家

- 机器学习工程师
- BI工程师
- 数据库管理员

### 3. 人工智能类岗位
- AI算法工程师
- 深度学习工程师
- 计算机视觉工程师
- 自然语言处理工程师
- AI产品经理
- AI研究员

### 4. 云计算与运维类岗位
- 云计算工程师
- DevOps工程师
- 系统架构师
- SRE工程师
- 网络工程师

### 5. 产品与设计类岗位

## 文件命名规范
文件采用"[岗位类别]-[专业级别].md"的命名格式：
- `development-junior.md` - 初级开发岗位清单
- `data-senior.md` - 高级数据岗位清单
- `ai-expert.md` - 专家级AI岗位清单
- `management-executive.md` - 管理层技术岗位清单

- 岗位定义和核心职责

- 职业发展前景分析

## 使用说明
1. **按类别查找**：根据感兴趣的岗位类别选择相应文件
2. **按级别筛选**：根据自身经验水平选择对应级别内容
3. **对比分析**：横向比较不同岗位的要求和发展前景
4. **职业规划**：结合个人情况制定合理的职业发展计划
