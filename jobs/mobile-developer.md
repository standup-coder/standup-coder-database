# 移动开发工程师职位描述(JD)

## 岗位基本信息
- **岗位名称**：移动开发工程师(Mobile Developer)
- **工作性质**：全职
- **工作地点**：北京/上海/深圳/杭州/成都
- **招聘人数**：若干
- **薪资范围**：18K-45K/月(根据经验水平)

## 岗位职责

### 核心开发工作
- 负责iOS/Android移动端应用的设计、开发和维护
- 实现用户界面和交互逻辑
- 优化移动应用性能和用户体验
- 保障代码质量和应用稳定性
- 参与产品需求分析和技术方案设计

### 平台适配优化
- 处理不同设备和系统版本的兼容性问题
- 优化应用启动速度和内存占用
- 实现推送通知和本地存储功能
- 集成第三方SDK和服务
- 监控应用性能和用户反馈

### 团队协作沟通
- 与UI/UX设计师协作完成功能实现
- 与后端工程师对接API接口
- 参与产品迭代和项目管理工作
- 分享移动端技术经验和最佳实践
- 参与代码评审和技术讨论

## 任职要求

### 必备技能
**iOS开发技能**：
- 精通Swift/Objective-C编程语言
- 熟悉iOS SDK和UIKit框架
- 掌握Auto Layout和界面适配
- 了解Core Data和网络编程
- 熟悉App Store发布流程

**Android开发技能**：
- 精通Java/Kotlin编程语言
- 熟悉Android SDK和Jetpack组件
- 掌握XML布局和Material Design
- 了解Room数据库和网络请求
- 熟悉Google Play发布流程

**通用技能**：
- 掌握Git版本控制工具
- 了解HTTP协议和RESTful API
- 具备良好的编码规范意识
- 具备性能优化和调试能力

### 经验要求
- **学历要求**：本科及以上学历，计算机相关专业
- **工作经验**：
  - 初级岗位：2-4年移动开发经验
  - 中级岗位：4-7年移动开发经验
  - 高级岗位：7年以上移动开发经验
- **项目经验**：有完整的移动应用开发和上线经验

### 加分技能
- 跨平台开发经验(React Native/Flutter)
- 混合开发经验(uni-app/Weex)
- 移动端架构设计经验
- 性能优化专家经验
- 开源项目贡献经历
