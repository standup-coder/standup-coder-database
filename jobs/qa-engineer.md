# 测试工程师职位描述(JD)

## 岗位基本信息
- **岗位名称**：测试工程师(QA Engineer)
- **工作性质**：全职
- **工作地点**：北京/上海/深圳/杭州/武汉
- **招聘人数**：若干
- **薪资范围**：12K-45K/月(根据经验水平)

## 岗位职责

### 功能测试
- 参与需求评审，制定测试计划和策略
- 编写测试用例，执行功能测试
- 提交和跟踪Bug，推动问题解决
- 输出测试报告，评估产品质量

### 自动化测试
- 设计和开发自动化测试脚本
- 搭建和维护自动化测试框架
- 推动自动化测试覆盖率提升
- 集成自动化测试到CI/CD流程

### 性能测试
- 制定性能测试方案和指标
- 使用性能测试工具进行压测
- 分析性能瓶颈，输出优化建议
- 建立性能基线，持续监控

### 质量保障
- 参与代码评审，发现潜在问题
- 推动研发团队提升代码质量
- 建立和完善质量保障流程
- 组织质量复盘，持续改进

## 任职要求

### 必备技能
**测试技术**：
- 熟练掌握功能测试方法和流程
- 熟悉接口测试(RESTful API)
- 掌握至少一种自动化测试框架
- 了解性能测试工具(JMeter/k6等)

**技术栈**：
- Python/Java/Go 至少一种编程语言
- Selenium/Appium 自动化测试
- Postman/JMeter 接口测试
- Git版本控制和Linux基础

### 经验要求
- **学历要求**：本科及以上学历
- **工作经验**：
  - 初级：1-3年测试经验
  - 中级：3-5年测试经验
  - 高级：5年以上测试经验

### 加分项
- 有自动化测试框架搭建经验
- 熟悉持续集成/持续部署流程
- 了解安全测试或渗透测试
- 有大数据或AI产品测试经验

## 薪酬福利

### 薪资结构
- **基本工资**：12K-45K/月
- **绩效奖金**：年度1-3个月薪资
- **质量奖金**：发现重大Bug奖励

### 福利待遇
- 五险一金 + 补充医疗保险
- 弹性工作时间
- 技术培训 + 测试大会参与

## 职业发展

### 发展路径
```
测试工程师 → 高级测试工程师 → 测试架构师 → 质量总监
          ↓
    专项测试专家(自动化/性能/安全)
```
