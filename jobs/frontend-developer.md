# 前端开发工程师职位描述(JD)

## 岗位基本信息
- **岗位名称**：前端开发工程师(Frontend Developer)
- **工作性质**：全职
- **工作地点**：北京/上海/深圳/杭州/成都
- **招聘人数**：若干
- **薪资范围**：18K-50K/月(根据经验水平)

## 岗位职责

### 前端开发实现
- 负责Web应用和移动端页面的开发
- 实现高保真的UI界面和流畅的交互体验
- 优化页面性能和加载速度
- 处理跨浏览器和跨设备兼容性问题
- 保障代码质量和可维护性

### 技术架构建设
- 参与前端技术选型和架构设计
- 建设前端组件库和工程化体系
- 优化构建流程和开发体验
- 推动前端性能监控体系建设
- 探索前沿前端技术和最佳实践

### 团队协作交付
- 与UI/UX设计师紧密配合实现设计稿
- 与后端工程师协作完成接口对接
- 参与需求评审和技术方案讨论
- 编写前端技术文档和开发规范
- 指导初级前端工程师成长

## 任职要求

### 必备技能
**核心基础**：
- 精通HTML5/CSS3/JavaScript(ES6+)
- 深入理解浏览器渲染原理和事件机制
- 掌握DOM操作和性能优化技巧
- 熟悉网络协议(HTTP/HTTPS/WebSocket)
- 了解前端安全常见问题和防范措施

**框架与工具**：
- 精通至少一个主流前端框架(React/Vue/Angular)
- 熟悉前端工程化工具(Webpack/Vite/Rollup)
- 掌握CSS预处理器(Sass/Less)和CSS-in-JS方案
- 熟悉前端测试框架(Jest/Vitest/Cypress)
- 了解TypeScript和类型系统

**工程能力**：
- 掌握Git版本控制和分支管理策略
- 熟悉前端性能优化手段和工具
- 具备组件化思维和设计能力
- 了解服务端渲染(SSR)和静态生成(SSG)
- 具备良好的代码审查能力

### 经验要求
- **学历要求**：本科及以上学历，计算机相关专业
- **工作经验**：
  - 初级岗位：2-4年前端开发经验
  - 中级岗位：4-7年前端开发经验
  - 高级岗位：7年以上前端架构经验
- **项目经验**：有大型单页应用或复杂交互系统开发经验

### 加分技能
- 跨平台开发经验(Flutter/React Native/Electron)
- 可视化开发经验(D3.js/ECharts/Canvas/WebGL)
- 低代码平台或无代码工具开发经验
- 开源项目贡献或技术社区活跃
- 全栈开发能力(Node.js/Python等)

## 薪酬福利

### 薪资结构
- **基本工资**：根据经验18K-50K/月
- **项目奖金**：产品上线和性能优化奖励
- **年终奖**：根据个人和团队表现
- **股票期权**：核心前端人员享有期权激励

### 福利待遇
- 五险一金 + 补充医疗保险
- 弹性工作时间 + 远程办公支持
- 年度体检 + 带薪年假
- 技术培训 + 设计工具订阅
- 团建活动 + 节日福利

## 职业发展

### 技术发展路径
```
Frontend Developer → Senior Frontend Developer → Frontend Expert
                ↓
          Tech Lead → Principal Engineer → Frontend Architect
```

### 管理发展路径
```
Senior Frontend Developer → Team Lead → Engineering Manager
```

### 专业细分方向
- **工程化专家**：构建工具、CI/CD、Monorepo
- **可视化专家**：数据可视化、图形渲染、WebGL
- **跨端专家**：小程序、RN、Flutter、桌面端
- **架构专家**：微前端、BFF、Serverless
