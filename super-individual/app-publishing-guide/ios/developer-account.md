# Apple 开发者账号注册

## 账号类型

| 类型 | 价格 | 人数 | App 上架 | 测试设备 | 支持 |
|------|------|------|-----------|----------|------|
| 个人 | $99/年 | 1人 | 可直接展示个人开发者名 | 100台 | 邮件支持 |
| 公司 | $299/年 | 多人在不同账号协作 | 显示公司名 | 100台/人 | 邮件支持 |

## 注册流程

### 1. 准备工作

```bash
# 必备条件
# - Mac 电脑（用于打包上传）
# - Apple ID（个人日常使用的即可）
# - 信用卡（Visa/MasterCard/JCB）
# - 身份证（个人）/ 营业执照（公司）

# 建议提前准备：
# 1. 确认 Apple ID 已开启双重认证
#    - 访问 https://appleid.apple.com
#    - 安全 → 双重认证 → 开启
# 2. 确认信用卡额度充足（$99 或 $299）
# 3. 确认身份证/营业执照在有效期内
```

### 2. 注册步骤

#### 第一步：访问注册页面

```bash
# 1. 打开浏览器访问：
#    https://developer.apple.com/programs/enroll/

# 2. 页面会显示所有可注册的账号类型：
#    - Apple Developer Program（iOS/macOS/watchOS 开发）
#    - Apple Developer Enterprise Program（企业内部分发）
#    - iOS Developer University Program（教育机构）

# 3. 选择适合你的类型，点击 "Enroll Now"
```

#### 第二步：选择账号类型

```bash
# 页面显示两种主要类型：

# 1. Individual（个人）
#    - 适合独立开发者
#    - 显示个人开发者名称
#    - 费用：$99/年

# 2. Company/Organization（公司）
#    - 适合团队或公司
#    - 显示公司名称
#    - 可添加多个团队成员
#    - 费用：$299/年

# 选择后点击 "Continue"
```

#### 第三步：确认身份（个人）

```bash
# 如果选择个人账号：
# 1. 页面显示 "Verify Your Identity"
# 2. 系统会自动关联你的 Apple ID
# 3. 需要验证：
#    - 姓名（必须与身份证一致）
#    - 地址
#    - 电话号码
#    - 身份证照片

# 上传身份证：
# 1. 点击 "Upload ID"
# 2. 上传身份证正反面照片
# 3. 照片需清晰可读
# 4. 支持 JPG/PNG/PDF 格式
```

#### 第三步：确认公司信息（公司）

```bash
# 如果选择公司账号：
# 1. 页面显示 "Legal Entity Status"

# 2. 填写公司信息：
#    - 公司法定名称（必须与营业执照一致）
#    - 简称
#    - 地址
#    - 网址（可选）

# 3. 邓白氏编码（D-U-N-S Number）
#    - 如果已有：直接输入
#    - 如果没有：点击 "Request a D-U-N-S Number" 申请
#    - 申请是免费的，但需要 1-2 周时间

# 4. 联系人信息：
#    - 主要联系人（将收到验证码）
#    - 联系电话
#    - 邮箱
```

#### 邓白氏编码申请（如需要）

```bash
# 如果公司没有邓白氏编码：
# 1. 点击 "Request a D-U-N-S Number"
# 2. 填写公司信息：
#    - 公司名称
#    - 主要营业地址
#    - 公司总机电话
#    - 业务类型
#    - 员工人数

# 3. 提交申请

# 4. 等待审核：
#    - 通常 1-2 个工作日
#    - 可能需要电话确认

# 5. 审核通过后获得编码
#    - 记录下这个编码，后续注册时需要
```

#### 第四步：验证邮箱和电话

```bash
# 1. 邮箱验证：
#    - 系统会发送验证邮件
#    - 打开邮件点击验证链接

# 2. 电话验证：
#    - 系统会拨打你的电话
#    - 接听并输入页面显示的验证码
```

#### 第五步：阅读并签署协议

```bash
# 1. 阅读 "Apple Developer Program License Agreement"
#    - 这是长达几十页的法律协议
#    - 建议完整阅读
#    - 了解 App Store 审核指南等重要内容

# 2. 勾选确认框：
#    - "I have read and agree to the Apple Developer Program License Agreement"
#    - "I acknowledge that I am bound by the Apple Developer Program License Agreement"

# 3. 点击 "Continue"
```

#### 第六步：购买会员资格

```bash
# 1. 页面跳转到购买页面
# 2. 确认购买信息：
#    - 个人：$99/年
#    - 公司：$299/年

# 3. 选择支付方式：
#    - 信用卡（推荐）
#    - 借记卡

# 4. 填写信用卡信息：
#    - 卡号
#    - 有效期
#    - CVV 码
#    - 持卡人姓名

# 5. 点击 "Purchase"

# 6. 等待扣款：
#    - 成功后会收到确认邮件
#    - 会员资格立即生效
```

#### 第七步：激活开发者账号

```bash
# 1. 回到 Apple Developer 网站
# 2. 使用刚注册的 Apple ID 登录
# 3. 页面显示 "You're a member"

# 4. 完善开发者资料（可选但推荐）：
#    - 头像
#    - 简介
#    - 联系方式

# 5. 访问 App Store Connect：
#    - https://appstoreconnect.apple.com/
#    - 使用同一 Apple ID 登录
#    - 可以开始创建 App
```

### 3. 邓白氏编码申请（公司账号）

如果公司没有邓白氏编码：

1. 访问 https://developer.apple.com/enroll/duns-lookup/
2. 点击 "Check D-U-N-S Number"
3. 输入公司信息查询
4. 如无编码，需先申请：
   - 免费，但需 1-2 周
   - 访问 https://www.dnb.com/duns-number/get-a-duns.html

### 4. 常见问题

**Q: 个人账号能转公司账号吗？**
A: 不能，需要重新注册。

**Q: 账号到期会怎样？**
A: App 会从 App Store 下架，无法更新。直到续费。

**Q: 可以退款吗？**
A: 很少见，Apple 政策严格。

**Q: 企业账号和公司账号区别？**
A: 企业账号（Enterprise）是内部部署用，不能上架 App Store。

## 注册后配置

### 开启双重认证

```bash
# 1. 访问 appleid.apple.com
# 2. 点击 "Sign In and Security"
# 3. 选择 "Two-Factor Authentication"
# 4. 按提示完成设置
```

### 访问 App Store Connect

访问 https://appstoreconnect.apple.com/ 确认账号已激活。

## 费用说明

| 项目 | 金额 |
|------|------|
| 个人账号 | $99 USD/年 |
| 公司账号 | $299 USD/年 |
| 自动续费 | 到期前 30 天 |

## 续费流程

```bash
# 1. 访问 developer.apple.com
# 2. 点击 "Account"
# 3. 点击 "Renewal"
# 4. 确认续费信息
```

## 注意事项

- 账号名称必须与 legal name 一致
- 身份证/营业执照必须在有效期内
- 一个 Apple ID 只能注册一个开发者账号
- 退款几乎不可能，慎重决定
