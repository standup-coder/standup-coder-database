# 中国互联网企业100强清单（2024年）

## 1. 主题介绍
本文件汇总了中国互联网协会发布的《中国互联网企业综合实力指数》百强榜单。该榜单是中国互联网行业最具权威性的企业排名之一，综合评估企业的营业收入、研发投入、用户规模、创新能力等多维度指标。2024年榜单于2024年10月17日在厦门发布，入围企业互联网业务收入总额达5.5万亿元。

## 2. 榜单概况

| 指标 | 数值 |
|------|------|
| 发布机构 | 中国互联网协会 |
| 发布时间 | 2024年10月17日 |
| 上榜企业总数 | 100家 |
| 互联网业务收入总额 | 5.5万亿元 |
| 研发投入总额 | 3200亿元+ |
| 员工总数 | 200万+ |

### 2.1 2024年榜单特点
- 收入利润实现双增长
- 研发投入力度同比加大
- 人工智能应用加速落地
- 出海业务成为新增长极
