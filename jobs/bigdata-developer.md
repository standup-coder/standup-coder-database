# 大数据开发工程师职位描述(JD)

## 岗位基本信息
- **岗位名称**：大数据开发工程师(Big Data Engineer)
- **工作性质**：全职
- **工作地点**：北京/上海/深圳/杭州/成都
- **招聘人数**：若干
- **薪资范围**：20K-55K/月(根据经验水平)

## 岗位职责

### 数据平台开发
- 设计和开发大数据处理平台
- 构建数据仓库和数据湖架构
- 开发ETL/ELT数据管道
- 优化数据处理性能和成本

### 实时计算
- 设计和开发实时数据流处理
- 使用Flink/Spark Streaming构建实时应用
- 开发实时数据监控和告警系统
- 保障实时数据质量和稳定性

### 数据治理
- 建立数据质量监控体系
- 制定数据标准和规范
- 开发数据血缘和元数据管理
- 保障数据安全和合规

### 技术支持
- 支持数据分析师和算法工程师
- 优化查询性能，提升用户体验
- 解决大数据平台技术问题
- 编写技术文档和最佳实践

## 任职要求

### 必备技能
**大数据技术栈**：
- Hadoop/Spark/Flink 生态
- Hive/Presto/ClickHouse 查询引擎
- Kafka/Pulsar 消息队列
- HBase/Elasticsearch 存储系统

**编程能力**：
- Java/Scala/Python 至少两种
- SQL 优化和复杂查询编写
- 熟悉分布式系统原理
- Linux 系统操作和Shell脚本

### 经验要求
- **学历要求**：本科及以上学历
- **工作经验**：
  - 初级：1-3年大数据经验
  - 中级：3-5年大数据经验
  - 高级：5年以上大数据经验

### 加分项
- 有PB级数据处理经验
- 熟悉云原生大数据产品(AWS EMR/阿里云MaxCompute等)
- 有数据湖(Data Lake)建设经验
- 熟悉数据治理框架

## 薪酬福利

### 薪资结构
- **基本工资**：20K-55K/月
- **绩效奖金**：年度1-4个月薪资
- **股票期权**：核心岗位享有期权

### 福利待遇
- 五险一金 + 补充医疗保险
- 弹性工作时间
- 大数据技术培训和大会

## 职业发展

### 发展路径
```
大数据工程师 → 高级大数据工程师 → 大数据架构师 → 数据平台总监
```
