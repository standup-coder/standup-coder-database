# 中国独角兽企业清单（2025年）

## 1. 主题介绍
本文件汇总了2025年中国独角兽企业榜单。独角兽企业是指成立于2000年之后、估值超过10亿美元（约70亿人民币）的非上市公司。2025年中国独角兽企业达372家，总估值突破1.2万亿美元。中国独角兽数量仅次于美国，位居全球第二，主要分布在人工智能、金融科技、电子商务、新能源、生物医药等领域。

## 2. 榜单概况

| 指标 | 数值 |
|------|------|
| 主要发布机构 | 胡润研究院、长城战略咨询 |
| 中国独角兽总数 | 372家 |
| 总估值 | 1.2万亿美元+ |
| 独角兽门槛 | 估值10亿美元以上 |
