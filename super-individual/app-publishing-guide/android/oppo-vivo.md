## 账号注册（OPPO & vivo）

### OPPO 注册详细步骤

#### 第一步：访问开放平台

```bash
# 1. 访问 https://open.oppommel.com/
# 2. 点击右上角 "登录"
# 3. 如果没有账号，点击 "注册"
```

#### 第二步：选择开发者类型

```bash
# 个人开发者：
# 1. 选择 "个人开发者"
# 2. 填写信息：
#    - 姓名
#    - 身份证号
#    - 手机号码
#    - 邮箱
# 3. 上传身份证照片
# 4. 等待审核（1-2 天）
```

#### 第三步：企业认证（如需）

```bash
# 1. 选择 "企业开发者"
# 2. 填写企业信息：
#    - 企业名称
#    - 统一社会信用代码
#    - 营业执照注册号
# 3. 上传营业执照
# 4. 填写法人信息
# 5. 填写联系人信息
# 6. 对公账户打款验证
# 7. 等待审核（2-3 天）
```

## 常见问题

**Q: OPPO 和 vivo 可以同时注册吗？**
A: 可以，分别注册即可。可通过小米平台一键同步。

**Q: 软著必须一样吗？**
A: 软著上的 App 名称需与提交的应用名称一致或包含关系。

**Q: 审核被拒了怎么办？**
A: 查看具体原因，修改后重新提交。必要时申诉。

**Q: 支持快应用吗？**
A: 支持，OPPO 和 vivo 都是快应用联盟成员。

**Q: 应用签名必须一样吗？**
A: 必须，后续更新也必须使用相同签名。
