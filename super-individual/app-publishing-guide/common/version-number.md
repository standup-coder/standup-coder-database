# 版号申请流程指南

## 什么是版号

版号（出版物号）是国内游戏/内容类 App 商业化必须获得的政府批准文号。

## 哪些需要版号

```bash
# 必须版号：
# - 网络游戏（端游、页游、手游）
# - 进口游戏
# - 付费下载的游戏/App
# - 包含内购的内容类 App

# 不必须版号（通常）：
# - 工具类 App
# - 纯免费无内购 App
# - 部分资讯类 App
```

## 审批部门

```bash
# 1. 国家新闻出版署
#    - 游戏版号审批
#    - 最高层级

# 2. 各省出版局
#    - 部分类别审批
#    - 进口游戏审批

# 3. 工业和信息化部
#    - ICP 备案
#    - App 备案
```

## 游戏版号申请

### 申请条件

```bash
# 1. 主体资质：
#    - 必须是企业
#    - 有网络文化经营许可证（部分情况）
#    - 有 ICP 备案

# 2. 游戏内容：
#    - 符合国家法律法规
#    - 无违规内容
#    - 完成防沉迷系统
#    - 实名认证系统
```

### 申请材料

```bash
# 1. 游戏介绍
#    - 游戏玩法介绍
#    - 游戏故事情节
#    - 游戏截图

# 2. 防护措施
#    - 防沉迷系统说明
#    - 实名认证系统说明
#    - 家长监护系统

# 3. 资质证明
#    - 企业营业执照
#    - 网络文化经营许可证
#    - 软件著作权证书

# 4. 审核申请表
#    - 游戏名称（中英文）
#    - 游戏类型
#    - 收费方式
```

### 申请流程

### 游戏版号申请详细步骤

#### 第一步：确认资质

```bash
# 确认是否需要版号：
# - 网络游戏（端游、页游、手游）：必须
# - 进口游戏：必须
# - 付费下载游戏：必须
# - 免费无内购游戏：部分需要

# 确认申请主体：
# - 必须是企业（公司）
# - 需要网络文化经营许可证（部分情况）
# - 需要 ICP 备案
```

#### 第二步：准备材料

```bash
# 1. 游戏介绍材料
#    - 游戏玩法介绍（详细文字说明）
#    - 游戏故事情节大纲
#    - 游戏操作说明
#    - 游戏截图（10-20 张）
#    - 游戏视频（1-3 分钟）

# 2. 防沉迷系统材料
#    - 防沉迷系统说明
#    - 实名认证系统说明
#    - 家长监护系统介绍

# 3. 企业资质
#    - 营业执照副本
#    - 网络文化经营许可证
#    - ICP 备案证明
#    - 软件著作权证书

# 4. 审核申请表
#    - 游戏名称（中英文）
#    - 游戏类型
#    - 收费方式（免费/付费/内购）
#    - 游戏题材
#    - 适龄提示
```

#### 第三步：提交申请

```bash
# 1. 访问审批系统
#    - 多数省份使用 "全国文化市场技术监管与服务平台"
#    - 或当地出版局指定的系统

# 2. 注册账号
#    - 使用企业信息注册
#    - 上传营业执照认证

# 3. 填写申请信息
#    - 按照表单逐项填写
#    - 上传所有材料

# 4. 缴纳审批费用（如有）
#    - 部分省份免费
#    - 部分省份收取费用

# 5. 提交申请
```

#### 第四步：等待审批

```bash
# 审批流程：
# 1. 省级出版局初审（1-2 个月）
#    - 材料完整性检查
#    - 内容合规性审查
# 2. 初审通过后上报（1 周）
# 3. 国家新闻出版署复审（3-4 个月）
#    - 内容终审
#    - 决定是否发放版号

# 总计：约 6 个月
```

#### 第五步：领取版号

```bash
# 审批通过后：
# 1. 在系统中下载版号批复文件
# 2. 版号批复包含：
#    - 版号（如：新广出审[2024]1234号）
#    - 游戏名称
#    - 申报企业
# 3. 在游戏中展示版号信息
```

```bash
# 1. 准备材料
#    - 6 个月内准备好全部材料

# 2. 提交申请
#    - 通过审批系统提交
#    - 缴纳审批费用

# 3. 初审
#    - 省级出版局初审
#    - 通过后上报

# 4. 复审
#    - 国家新闻出版署复审

# 5. 发放版号
#    - 审批通过后发放
```

### 审批周期

```bash
# 正常周期：
# - 省级审核：1-2 个月
# - 国家级审核：3-4 个月
# - 总计：约 6 个月

# 加急（不保证）：
# - 部分代理声称可加急
# - 费用较高
```

### 费用

```bash
# 官方费用：免费

# 代理费用（如需要）：
# - 普通代理：3000-5000 元
# - 加急代理：8000-15000 元

# 注意：
# - 版号申请难度大
# - 建议找专业代理
```

## App 备案（2023年起）

### 备案背景

```bash
# 工信部要求：
# - 国内 App 需进行备案
# - 类似网站的 ICP 备案
# - 2023 年开始分阶段实施
```

### 备案流程

```bash
# 1. 准备材料：
#    - 企业营业执照
#    - App 负责人信息
#    - App 基本信息
#    - 隐私政策
#    - 软件著作权

# 2. 提交备案：
#    - 通过工信部备案系统
#    - 填写 App 信息
#    - 上传材料

# 3. 审核：
#    - 省级通信管理局审核
#    - 周期：1-2 个月

# 4. 备案号：
#    - 获得备案号
#    - 需在 App 中展示
```

## ICP 备案

### 什么是 ICP 备案

```bash
# ICP 备案：
# - 国内网站/App 服务器的必要资质
# - 工信部颁发
# - 证明网站合法运营
```

### 备案要求

```bash
# 1. 主体：
#    - 企业或个体户
#    - 个人网站（部分限制）

# 2. 材料：
#    - 营业执照（企业）
#    - 法人身份证
#    - 域名证书
#    - 服务器租赁合同

# 3. 流程：
#    - 服务商代备案
#    - 或直接通过工信部
```

### 备案号展示

```bash
# App 中需展示：
# - ICP 备案号
# - 链接到工信部查询

# 格式：
# 京ICP备XXXXXXXX号-1
```

## 合规建议

### 超级个体应对策略

```bash
# 1. 早期验证阶段：
#    - 先做微信小程序
#    - 不需要版号
#    - 快速验证产品

# 2. 海外优先：
#    - 先上 Google Play
#    - 海外版不需要国内版号
#    - 验证商业模式

# 3. 工具类切入：
#    - 优先做工具类 App
#    - 软著即可发布
#    - 避免版号需求

# 4. 后续扩展：
#    - 产品验证后补齐资质
#    - 申请软著
#    - 考虑版号申请
```

### 风险规避

```bash
# 1. 不要先上国内版：
#    - 无版号可能面临下架
#    - 法律风险

# 2. 不要轻视：
#    - 政策趋严
#    - 处罚力度加大

# 3. 合规运营：
#    - 及时备案
#    - 资质齐全
#    - 内容合规
```

## 常见问题

**Q: 软著和版号可以同时申请吗？**
A: 可以，两者独立审批。

**Q: 个人开发者需要版号吗？**
A: 理论上需要企业主体，建议咨询专业机构。

**Q: 版号申请被拒怎么办？**
A: 根据审批意见修改后重新申请。

**Q: 购买已有版号的游戏可以吗？**
A: 可以通过转让或授权方式，但要确保合法。

**Q: App 备案和 ICP 备案一样吗？**
A: 不一样，App 备案是针对移动应用的新要求。

**Q: 海外游戏需要版号吗？**
A: 进口游戏需要，且审批更严格。
