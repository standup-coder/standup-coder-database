# UI/UX设计师职位描述(JD)

## 岗位基本信息
- **岗位名称**：UI/UX设计师(UI/UX Designer)
- **工作性质**：全职
- **工作地点**：北京/上海/深圳/杭州/成都
- **招聘人数**：若干
- **薪资范围**：15K-50K/月(根据经验水平)

## 岗位职责

### UI设计
- 负责产品界面视觉设计，输出高保真设计稿
- 制定和维护设计规范(Design System)
- 设计图标、插画等视觉元素
- 跟进设计落地，确保还原度

### UX设计
- 进行用户研究和竞品分析
- 设计用户流程和信息架构
- 制作线框图和交互原型
- 组织可用性测试，收集用户反馈

### 协作交付
- 与产品经理沟通需求，理解业务目标
- 与前端工程师协作，确保设计可实现
- 输出设计规范和资源给开发团队
- 参与设计评审，持续优化体验

## 任职要求

### 必备技能
**设计工具**：
- Figma/Sketch/Adobe XD 界面设计
- Photoshop/Illustrator 视觉设计
- Principle/Framer/ProtoPie 交互动效
- Axure 原型设计

**设计能力**：
- 优秀的视觉设计能力和审美
- 了解用户体验设计原则
- 熟悉iOS/Android/Web设计规范
- 具备设计系统构建能力

### 经验要求
- **学历要求**：设计相关专业本科及以上
- **工作经验**：
  - 初级：1-3年设计经验
  - 中级：3-5年设计经验
  - 高级：5年以上设计经验
- **作品集**：需提供完整作品集

### 加分项
- 有知名互联网产品设计经验
- 具备动效设计或3D设计能力
- 了解前端开发基础知识
- 有设计系统建设经验

## 薪酬福利

### 薪资结构
- **基本工资**：15K-50K/月
- **绩效奖金**：年度1-4个月薪资
- **项目奖金**：设计创新奖励

### 福利待遇
- 五险一金 + 补充医疗保险
- Mac Pro/iMac 专业设备
- 设计软件授权报销
- 设计大会参与机会

## 职业发展

### 发展路径
```
UI设计师 → 高级设计师 → 设计专家 → 设计总监
       ↓
UX研究员 → 交互设计师 → 体验设计师 → 体验架构师
```
