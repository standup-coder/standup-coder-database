# 网络工程师职位描述(JD)

## 岗位基本信息
- **岗位名称**：网络工程师(Network Engineer)
- **工作性质**：全职
- **工作地点**：北京/上海/深圳/杭州/成都
- **招聘人数**：若干
- **薪资范围**：20K-45K/月(根据经验水平)

## 岗位职责

### 网络架构设计
- 负责企业网络架构的规划和设计
- 制定网络设备选型和部署方案
- 优化网络性能和安全性配置
- 设计网络监控和管理策略
- 制定网络应急预案和灾备方案

### 网络运维管理
- 负责网络设备的日常运维和监控
- 处理网络故障和性能问题
- 实施网络安全防护措施
- 管理网络配置和变更控制
- 优化网络资源利用效率

### 技术支持协作
- 为业务部门提供网络技术支持
- 协助其他团队解决网络相关问题
- 参与网络项目的技术评估
- 分享网络最佳实践和经验
- 推动网络技术创新和应用

## 任职要求

### 必备技能
**网络技术**：
- 精通TCP/IP协议栈和网络原理
- 熟悉主流网络设备厂商(Cisco/华为/H3C)
- 掌握路由交换技术(OSPF/BGP/MPLS)
- 熟悉网络安全技术(防火墙/IDS/VPN)
- 掌握网络监控工具(Nagios/Zabbix)

**实践能力**：
- 具备网络架构设计能力
- 熟悉网络故障排查方法
- 具备网络安全防护经验
- 了解SDN和网络虚拟化技术
- 具备自动化运维能力

### 经验要求
- **学历要求**：本科及以上学历，计算机/网络工程相关专业
- **工作经验**：
  - 初级岗位：3-5年网络运维经验
  - 中级岗位：5-8年网络工程经验
  - 高级岗位：8年以上网络架构经验
- **项目经验**：有大型网络建设项目经验

### 加分技能
- CCIE/HCIE等高级网络认证
- 云网络架构设计经验
- 网络安全专家经验
- 自动化运维工具开发经验
- 开源网络项目贡献经历

- **网络设备**：Cisco + 华为 + H3C

- **安全设备**：Palo Alto + Fortinet + Check Point
- **自动化**：Ansible + Python + Netmiko
- **云网络**：AWS VPC + Azure Virtual Network

- 定期技术分享和最佳实践
