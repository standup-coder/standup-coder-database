# 无锡云计算企业清单

## 1. 主题介绍
云计算是一种通过网络提供计算资源的服务模式，包括基础设施即服务(IaaS)、平台即服务(PaaS)和软件即服务(SaaS)。无锡作为中国物联网产业的重要基地，拥有多家专注于云计算领域的企业，尤其在工业云、物联网云等方向具有特色。
