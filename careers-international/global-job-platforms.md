# 国际主要招聘平台清单

## 1. 概述

本清单收录全球主要的国际化招聘平台、求职网站和相关信息来源，为海外求职者、留学生、国际化人才提供全面的求职参考。

## 2. 综合招聘平台

### 2.1 LinkedIn(领英)
- **官网**：[https://www.linkedin.com](https://www.linkedin.com)
- **成立时间**：2002年
- **总部**：美国硅谷
- **特点**：全球最大的职业社交平台
- **主要功能**：职业档案、人脉拓展、职位搜索、行业洞察
- **适用人群**：国际化人才、外企求职者、海外留学生
- **数据特色**：全球用户8亿+，覆盖200+国家和地区

### 2.2 Indeed
- **官网**：[https://www.indeed.com](https://www.indeed.com)
- **成立时间**：2004年
- **总部**：美国奥斯汀
- **特点**：全球最大求职搜索引擎
- **主要功能**：职位聚合搜索、薪资比较、公司评价、求职指导
- **适用人群**：全球求职者，特别适合英语求职者
- **数据特色**：每月访问用户2.5亿+，聚合全球数千万职位

### 2.3 Glassdoor
- **官网**：[https://www.glassdoor.com](https://www.glassdoor.com)
- **成立时间**：2007年
- **总部**：美国旧金山
- **特点**：员工匿名评价雇主，透明度高
- **主要功能**：公司评价、薪资信息、面试经验、办公环境
- **适用人群**：求职者了解公司真实情况、在职员工
- **数据特色**：收录公司评价1000万+条，薪资信息500万+条

### 2.4 Monster
- **官网**：[https://www.monster.com](https://www.monster.com)
- **成立时间**：1994年
- **总部**：美国威斯康星州
- **特点**：老牌国际招聘网站
- **主要功能**：职位搜索、简历管理、职业指导、薪资调查
- **适用人群**：全球求职者，特别适合北美市场
- **数据特色**：服务企业客户超过40万家

### 2.5 Seek
- **官网**：[https://www.seek.com.au](https://www.seek.com.au)
- **成立时间**：1997年
- **总部**：澳大利亚墨尔本
- **特点**：澳洲最大招聘网站
- **主要功能**：职位搜索、简历投递、职业建议、薪资调查
- **适用人群**：澳洲求职者、新西兰求职者
- **数据特色**：澳洲市场份额超过60%，覆盖新西兰市场

### 2.6 Reed.co.uk
- **官网**：[https://www.reed.co.uk](https://www.reed.co.uk)
- **成立时间**：1966年
- **总部**：英国伦敦
- **特点**：英国老牌招聘网站
- **主要功能**：职位搜索、简历服务、职业培训、薪资调查
- **适用人群**：英国求职者、爱尔兰求职者
- **数据特色**：英国市场份额领先，历史悠久信誉良好

## 3. 垂直领域招聘平台

- **官网**：[https://stackoverflow.com/jobs](https://stackoverflow.com/jobs)
- **成立时间**：2010年
- **总部**：美国纽约
- **特点**：专注技术人才招聘
- **主要功能**：程序员职位、技术社区、薪资调研、远程工作
- **适用人群**：软件开发人员、技术专家
- **数据特色**：技术岗位占比90%+，薪资信息透明度高

### 3.2 GitHub Jobs

- **成立时间**：2008年
- **总部**：美国旧金山

- **主要功能**：技术职位、开源项目、团队文化展示
- **适用人群**：程序员、开源贡献者、技术团队
- **数据特色**：高质量技术岗位，注重开源文化

- **官网**：[https://angel.co](https://angel.co)
- **成立时间**：2010年
- **总部**：美国旧金山
- **特点**：专注初创企业和天使投资
- **主要功能**：初创公司职位、股权激励、投资人对接
- **适用人群**：创业者、早期员工、投资人
- **数据特色**：覆盖全球初创企业20万+，投资机会丰富

### 3.4 Honeypot
- **官网**：[https://www.honeypot.io](https://www.honeypot.io)
- **成立时间**：2015年
- **总部**：德国柏林

- **主要功能**：技术职位匹配、技能评估、薪资谈判
- **适用人群**：欧洲技术人才、跨国公司
- **数据特色**：专注欧洲市场，技术岗位匹配精准

### 3.5 We Work Remotely
- **官网**：[https://weworkremotely.com](https://weworkremotely.com)
- **成立时间**：2013年
- **总部**：美国旧金山
- **特点**：专注远程工作机会
- **主要功能**：远程职位、分布式团队、灵活工作安排
- **适用人群**：远程工作者、数字游民、自由职业者
- **数据特色**：远程工作职位5万+，覆盖全球企业

## 9. 使用建议

### 9.1 国际化求职策略
- **多平台布局**：根据目标国家选择当地主流平台
- **语言准备**：英语是国际通用语言，但需了解当地语言要求
- **文化适应**：研究目标国家的职场文化和招聘习惯
- **资质认证**：确认学历和职业资格的国际认可度

### 9.2 平台选择建议
- **北美市场**：LinkedIn、Indeed、Glassdoor为主
- **欧洲市场**：LinkedIn、Monster、Honeypot为辅
- **亚太市场**：LinkedIn、Seek、当地平台结合使用
- **技术岗位**：Stack Overflow Jobs、GitHub Jobs优先考虑

### 9.3 风险防范
- **信息核实**：通过官方渠道验证公司和职位真实性
- **合同审查**：仔细阅读雇佣合同条款，必要时寻求法律帮助
- **签证办理**：提前了解工作签证要求和办理流程
- **税务规划**：了解海外收入的税务处理方式
