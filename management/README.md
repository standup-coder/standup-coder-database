# 技术管理知识体系

## 概述
本目录包含完整的技术管理知识体系，涵盖从基础框架到专项实践的全方位管理指导。适用于技术团队负责人、工程管理者、CTO等各级技术管理者。

## 目录结构

### 📊 核心框架 (core-framework)
技术管理的基础理论和核心框架，提供管理体系建设的根本指导。

### 🎯 专项指南 (specialized-guides)  
针对特定领域和场景的管理实践指南，包括新兴技术和管理模式。

### 📊 评估工具 (assessment-tools)
管理能力评估体系和工具，帮助识别管理优势和改进方向。

### 📚 支持文档 (support-documents)
项目管理、质量标准、整合指南等支持性文档。

## 使用建议

### 入门路径
1. 从 **core-framework** 开始了解基础管理框架
2. 使用 **assessment-tools** 进行能力自评
3. 根据实际需求参考 **specialized-guides** 中的相关指南
4. 查阅 **support-documents** 获取项目管理和质量标准信息

### 进阶应用
- 结合多个专项指南解决复杂管理问题
- 建立组织内部的管理评估和改进机制
- 将理论框架与实际业务场景相结合

## 文档维护
- 定期更新最新管理实践和工具
- 保持文档间的链接和引用关系
- 收集用户反馈持续优化内容
